setupアカウントの作成方法

① 初めてアカウントを作成する場合

ユーザー名とパスワードを設定する画面が表示されたら、

ユーザー名 : [ setup ]

パスワード : [苗字の頭文字 + 学籍番号]

パスワードのヒント : [　(半角スペース)　]

これらを入力して「次へ」をクリックします。



② すでに他のアカウントがPCに存在する場合

ユーザー名とパスワードを設定する画面が表示されたら、

ユーザー名 : [ setup ]

パスワード : なし

※ここではパスワードは設定しません

〜アカウント作成を最後まで進めます〜

デスクトップ画面が表示されたらスタートボタンを右クリックします

一覧からコントロールパネルを選択します

ユーザーアカウントをクリックします

もう一度、ユーザーアカウントをクリックします

PC設定でアカウントを変更をクリックします

サインインオプションをクリックします

ユーザーアカウントをクリックします

パスワード : [苗字の頭文字 + 学籍番号]

パスワードのヒント : [　(半角スペース)　]

これらを入力して「次へ」をクリックします。
