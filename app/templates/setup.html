setupアカウントの作成方法

setupアカウントは非常用のため作成をお願いします。通常は使用しないでください。

※windows10のsetupアカウント作成手順です。Macの場合はPCクリニックに相談してください。

①他に作成済みのアカウントがない場合

ユーザー名とパスワードを設定する画面が表示されたら、

ユーザー名 : [ setup ]

パスワード : なし

※ここではパスワードは設定しません

〜アカウント作成を最後まで進めます〜

デスクトップ画面が表示されたらスタートボタンを右クリックし、設定を選択します。

アカウントをクリックします。

サインインオプションをクリックします。

パスワードをクリックして追加ボタンをクリックします。

パスワード : [苗字の頭文字 + 学籍番号]

パスワードのヒント : [　(半角スペース)　]

パスワード設定まで完了しました。次に管理者アカウントの作成を行います。


元の画面に戻り家族とその他のユーザーをクリックします。

その他のユーザーをこのPCに追加をクリックします。

ユーザー名、パスワード、パスワードを忘れた場合のヒントを入力します。

家族とその他ユーザーの画面に戻り作成した自分のアカウントをクリックします。

アカウントの種類の変更ボタンをクリックし、管理者を選択します。

管理者に変更できたことを確認したら最後にPCを再起動します。

以上で設定完了です。



②すでに他に作成済みのアカウントがある場合

デスクトップ画面が表示されたらスタートボタンを右クリックし、設定を選択します。

アカウントをクリックします。

その他のユーザーをこのPCに追加をクリックします。

ユーザー名、パスワード、パスワードを忘れた場合のヒントを入力します。

ユーザー名 : [ setup ]

パスワード : [苗字の頭文字 + 学籍番号]

パスワードのヒント : [　(半角スペース)　]

家族とその他ユーザーの画面に戻り作成したsetupアカウントをクリックします。

アカウントの種類の変更ボタンをクリックし、管理者を選択します。

管理者に変更できたことを確認したら最後にPCを再起動します。

以上で設定完了です。
