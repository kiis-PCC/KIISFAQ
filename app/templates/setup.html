setupアカウントの作成方法

setupアカウントは非常用のため作成をお願いします。通常は使用しないでください。

※windows10のsetupアカウント作成手順です。Macの場合はPCクリニックに相談してください。

  • 他に作成済みのアカウントがない場合
  • すでに他に作成済みのアカウントがある場合

他に作成済みのアカウントがない場合

①.アカウント設定

    ~まずは初期設定を進めます~

  1. ユーザー名とパスワードを設定する画面が表示されたら、ユーザー名 : [ setup ]　と入力し「次へ」をクリックします。

    2. パスワード : なし

    ※何も入力せず「次へ」をクリックします。（パスワードは後で設定します。）

    〜アカウント作成を最後まで進めます〜

    ②.管理者アカウントの追加とパスワード設定

  2. デスクトップ画面が表示されます。
  3. スタートボタンをクリックし、設定ボタンをクリックします。
  4. Windowsの設定画面が表示されます。アカウントをクリックします。
  5. まずはsetupアカウントのパスワードを設定します。サインインオプションをクリックします。
  6. パスワードをクリックして追加ボタンをクリックします。
  7. パスワードの作成画面が表示されます。 パスワード入力欄に「苗字のアルファベット最初の一文字＋学籍番号」、 パスワードのヒント欄に「 」（空白文字を一文字）を入力し、 次へボタンをクリックします。
  8. 完了ボタンをクリックします。これでsetupアカウントのパスワードの設定は完了です。
  9. 次に管理者アカウントの作成を行います。
  10. 元の画面に戻り家族とその他のユーザーをクリックします。
  11. その他のユーザーをこのPCに追加をクリックします。
  12. ユーザー名、パスワード、パスワードを忘れた場合のヒントを入力し、次へをクリックします。
  13. （ヒントに関して：画像は例です。実際は自分で分かりやすい分かりやすい質問と答えを設定するようにしてください）
  14. 家族とその他ユーザーの画面に戻るので作成した自分のアカウント（例ではTaro）をクリックし、
  15. アカウントの種類の変更ボタンをクリックします。
  16. 「管理者」を選択して、OKボタンをクリックします。
  17. 管理者に変更できたことを確認したら最後にPCを再起動します。

    18. 以上で設定完了です。



すでに他に作成済みのアカウントがある場合

  1. デスクトップ画面でスタートボタンを右クリックし、設定を選択します。
  2. Windowsの設定画面が表示されます。アカウントをクリックします。
  3. その他のユーザーをこのPCに追加をクリックします。
  4. ユーザー名、パスワード、パスワードを忘れた場合のヒントを入力します。

    5. ユーザー名 : [ setup ]

    パスワード : [苗字の頭文字 + 学籍番号]

    パスワードのヒント : [　(半角スペース)　]

  5. 家族とその他ユーザーの画面に戻るので作成した「setup」をクリックし、アカウントの種類の変更ボタンをクリックします。
  6. 「管理者」を選択して、OKボタンをクリックします。
  7. 管理者に変更できたことを確認したら最後にPCを再起動します。

    8. 以上で設定完了です。
