setupアカウントの作成方法

setupアカウントは非常用のため作成をお願いします。通常は使用しないでください。

※windows10のsetupアカウント作成手順です。Macの場合はPCクリニックに相談してください。

①他に作成済みのアカウントがない場合

1.アカウント設定

まずは初期設定を進めます。

ユーザー名とパスワードを設定する画面が表示されたら、ユーザー名 : [ setup ]　と入力し「次へ」をクリックします。

パスワード : なし

※何も入力せず「次へ」をクリックします。（パスワードは後で設定します。）

〜アカウント作成を最後まで進めます〜

2.管理者アカウントの追加とパスワード設定

デスクトップ画面が表示されます。

スタートボタンをクリックし、設定ボタンをクリックします。

Windowsの設定画面が表示されます。アカウントをクリックします。

まずはsetupアカウントのパスワードを設定します。サインインオプションをクリックします。

パスワードをクリックして追加ボタンをクリックします。

パスワードの作成画面が表示されます。 パスワード入力欄に「苗字のアルファベット最初の一文字＋学籍番号」、 パスワードのヒント欄に「 」（空白文字を一文字）を入力し、 次へボタンをクリックします。

完了ボタンをクリックします。これでsetupアカウントのパスワードの設定は完了です。

パスワード設定まで完了しました。次に管理者アカウントの作成を行います。


元の画面に戻り家族とその他のユーザーをクリックします。

その他のユーザーをこのPCに追加をクリックします。

ユーザー名、パスワード、パスワードを忘れた場合のヒントを入力します。

家族とその他ユーザーの画面に戻り作成した自分のアカウントをクリックします。

アカウントの種類の変更ボタンをクリックし、管理者を選択します。

管理者に変更できたことを確認したら最後にPCを再起動します。

以上で設定完了です。



②すでに他に作成済みのアカウントがある場合

デスクトップ画面が表示されたらスタートボタンを右クリックし、設定を選択します。

アカウントをクリックします。

その他のユーザーをこのPCに追加をクリックします。

ユーザー名、パスワード、パスワードを忘れた場合のヒントを入力します。

ユーザー名 : [ setup ]

パスワード : [苗字の頭文字 + 学籍番号]

パスワードのヒント : [　(半角スペース)　]

家族とその他ユーザーの画面に戻り作成したsetupアカウントをクリックします。

アカウントの種類の変更ボタンをクリックし、管理者を選択します。

管理者に変更できたことを確認したら最後にPCを再起動します。

以上で設定完了です。
