ネットワークドライブの割り当て

本学では、1GBの容量を持つ、ネットワークドライブが利用できます。
ここでは、パソコンにこのネットワークドライブをS ドライブとして割り当てる方法を 説明します。S ドライブは、学内のどの場所からでも、どのパソコンからでも利用できます。 有意義に活用するようにしてください。
また、S ドライブはログオンしたユーザー専用のドライブです。 他の人との共有は出来ません。

Windows OS

  1. スタートボタンを右クリックし、エクスプローラーをクリックします。
  2. PCを右クリックし、ネットワークドライブの割り当てをクリックします。
  3. 「ドライブ」に「S:」を選択、「フォルダ」に「\\svr260007\std\z学籍番号」を入力し、完了ボタンをクリックします。
  4. 認証画面が表示されたら、kiisnw\z学籍番号・配布されたパスワードを入力し、OKボタンをクリックします。
  5. 認証に成功すると、自動的に S ドライブの中身が表示されます。

Mac OS

  1. デスクトップ左上のメニューバーから移動をクリックし、サーバーへ接続をクリックします。
  2. サーバアドレスに「smb://svr260007/share」を入力し、接続をクリックします。
  3. 名前とパスワードを入力するように求められるので、z学籍番号 パスワードを入力して接続をクリックします。
  4. 認証に成功したら、自動的に中身が表示されます。
